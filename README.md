# Features
1. 讲英文段落拆分，分别解释每个句子的语法结构
2. 选择句子中的单词，获取单词的信息
3. 历史查询过的句子和单词
4. 自定义保存句子和单词的数量
5. 保存句子
6. 保存单词
7. alfred插件支持
